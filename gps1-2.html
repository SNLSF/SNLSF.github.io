What is pair programming?

An approach to development involving two software engineers collaborating on a single piece of code with the goal of minimizing the number of bugs present in the final product. Why is pair programming important to DBC?

DBC is an institution which provides aspiring developers with the experience and technical skills required to succeed in a professional software or website development environment. Accordingly, as the pair programming methodology is ubiquitous across professional development verticals, DBC trains its students in this methodolgy from the very outset of the program. Compare Driver and Navigator

The DRIVER: this engineer helms the hardware and types in the actual code being developed.
